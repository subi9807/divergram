## 1. 사업 개요

Divergram은 스쿠버다이빙, 프리다이빙, 스노클링 이용자를 위한 다이빙 특화 플랫폼이다. 기존의 SNS 기능에 더해 다이빙 기록 관리, 해양 날씨 확인, 장비 로그 연동, 안전 경고, AI 보조 기능을 통합하여, 사용자가 다이빙 전후의 핵심 정보를 한 곳에서 관리할 수 있도록 설계했다.

본 사업은 단순한 커뮤니티 서비스가 아니라, 다이버의 안전 확보, 해양관광 활성화, 지역 다이빙 생태계 디지털화, 장비 데이터 자산화라는 공공성을 함께 갖는다. 이러한 이유로 민간 자본뿐 아니라 공공기관의 초기 투자 및 실증 지원과도 높은 적합성을 가진다.

## 2. 사업 추진 배경

다이빙 산업과 해양 레저 시장은 성장 가능성이 있으나, 실제 사용자 경험은 매우 분절적이다.

- 다이빙 기록은 장비 제조사별 앱이나 수기 메모에 흩어져 있다.
- 입수 전 해양 상태 확인은 외부 사이트를 따로 확인해야 한다.
- 지역 다이빙 포인트 정보는 표준화되어 있지 않다.
- 안전 판단은 개인 경험에 많이 의존한다.
- 콘텐츠 공유와 기록 보관이 분리되어 있어 재방문과 지속 사용이 어렵다.

이 구조는 사용자에게 불편을 줄 뿐 아니라, 해양 안전 정보의 활용도와 지역 관광의 데이터화도 제한한다. Divergram은 이러한 문제를 통합적으로 해결하는 디지털 인프라 역할을 목표로 한다.

## 3. 사업 목적

### 3.1 사용자 측면

- 다이빙 기록을 쉽고 정확하게 남긴다.
- 다이빙 전 해양 조건을 빠르게 확인한다.
- 장비 로그를 자동으로 가져와 기록 누락을 줄인다.
- 공유용 콘텐츠를 간편하게 생성한다.

### 3.2 공공 측면

- 해양 레저 안전 정보의 접근성을 높인다.
- 다이빙 포인트 및 지역 해양관광 정보의 디지털 전환을 촉진한다.
- 지역 다이빙샵, 리조트, 교육기관의 홍보 및 연결을 돕는다.
- 해양 레저 이용자의 안전 의사결정을 지원한다.

### 3.3 산업 측면

- 장비 제조사, 다이빙샵, 교육기관, 관광지와의 연계를 확대한다.
- 다이빙 로그와 위치, 날씨, 장비 데이터를 구조화하여 부가가치를 만든다.
- 향후 구독형 서비스, B2B 제휴, 지역 관광 연계 사업으로 확장한다.

## 4. 해결하려는 문제

### 4.1 사용자 문제

- 기록 관리가 번거롭고 지속성이 낮다.
- 로그와 사진, 위치, 장비 정보가 따로 관리된다.
- 다이빙 전 안전 판단에 필요한 정보가 분산되어 있다.
- 외부 장비 데이터를 수동으로 옮겨야 한다.

### 4.2 지역 및 산업 문제

- 다이빙 포인트와 해양 정보가 표준화되어 있지 않다.
- 지역 다이빙샵과 관광 정보가 파편화되어 있다.
- 해양 레저 관련 콘텐츠가 일회성 소비로 끝나기 쉽다.

### 4.3 공공정책 측면의 문제

- 해양 레저 안전 정보를 일상적으로 접하기 어렵다.
- 공공성 있는 디지털 서비스가 부족하다.
- 지역 기반 해양관광 인프라와 이용자 데이터를 연결할 통합 플랫폼이 부족하다.

## 5. 해결 방식

Divergram은 아래 4가지 축으로 문제를 해결한다.

### 5.1 기록 통합

- DiveLog 관리
- 사진, 영상, 메모, 장비, 버디, 태그, 공개 범위 편집
- Google Maps 기반 위치 연결
- 자격증 및 활동 이력 관리

### 5.2 안전 정보 통합

- Stormglass API 기반 해양 날씨 조회
- 파고, 조류, 수온, 풍속, 시야, 조수 시간, 기온 정보 제공
- 위험도 판정 및 초보자 경고 제공
- 추천 다이빙 시간대 제안

### 5.3 장비 로그 통합

- Garmin, Suunto, Shearwater Cloud 외부 연동
- Bluetooth 기반 다이빙 컴퓨터 로그 동기화
- 중복 방지, 재시도, 동기화 상태 관리

### 5.4 콘텐츠 및 확산

- 공유용 이미지 및 캡션 생성
- Instagram 공유 중심 외부 확산
- OpenAI 기반 요약 및 추천 문구 제공

## 6. 서비스 구성

### 6.1 모바일 앱

사용자 핵심 서비스로서 다음 기능을 제공한다.

- 피드, 릴스, 프로필
- 다이빙 포인트 지도
- DiveLog 목록 및 편집
- 해양 날씨 화면
- 외부 연동 및 Bluetooth 동기화
- 설정, 알림, 자격증, 정책 동의

### 6.2 관리자 웹

운영 안정성과 공공기관 협업을 위한 관리 콘솔이다.

- 회원 및 콘텐츠 관리
- 신고 및 제재 관리
- 자격증 및 인증 운영
- 공지 및 통계 확인

### 6.3 공개 웹

서비스 홍보, 사용자 유입, 기관 협업 안내를 담당한다.

- 서비스 소개
- 기능 안내
- 정책 및 고객지원
- 파트너/제휴 문의

### 6.4 API 서버

인증, 데이터, 연동, 푸시, 업로드를 담당한다.

- Express/PostgreSQL 기반
- 토큰 및 인증 관리
- 미디어 업로드
- 외부 API 동기화
- 푸시 알림 발송

## 7. 공공기관 투자 적합성

본 사업은 공공기관이 지원하기에 다음의 장점을 가진다.

### 7.1 공공성

- 해양 레저 안전 정보 제공
- 지역 관광 정보 디지털화
- 다이버 커뮤니티의 안전한 이용 지원

### 7.2 실증 가능성

- 앱, 관리자, API 구조가 분리되어 있어 단계적 실증이 가능하다.
- 해양 날씨, 위치, 장비 연동 등은 실제 사용자 데이터를 통해 성과 확인이 쉽다.

### 7.3 지역경제 연계

- 다이빙샵, 리조트, 강사, 장비 판매처와 연결 가능
- 지역 다이빙 포인트의 노출과 재방문 유도 가능

### 7.4 정책 연계성

- 해양 레저 안전
- 지역관광 활성화
- 디지털 전환
- 청년 창업 및 콘텐츠 산업

## 13. 재원 활용 계획

공공기관 투자금은 다음 항목에 우선 배분하는 것이 적절하다.

### 13.1 기술 개발

- BLE 및 외부 API 동기화
- 안정성 개선 및 테스트

### 13.2 공공성 강화

- 안전 경고 로직 개선
- 정책/동의/신고 체계 고도화
- 지역 포인트 및 관광 정보 구조화

### 13.3 운영 및 실증

- 파일럿 사용자 운영
- 기관 협업 자료 제작
- 사용자 피드백 반영

### 13.4 콘텐츠 및 확산

- 서비스 소개 자료
- 공유형 콘텐츠 생성
- 기관 제출용 보고서 및 대시보드

### 13.5 투자 요청 개요

공공기관 제출 시 아래 항목은 기관 양식에 맞춰 수치만 채워 넣을 수 있도록 설계한다.

- 요청 금액: `[예: 총 사업비 중 공공 투자 요청액 입력]`
- 자부담 비율: `[예: 현금/현물 포함 비율 입력]`
- 지원 형태: `[직접 투자 / 보조금 / 실증사업 / PoC 지원 / 융자 등 입력]`
- 사용 기간: `[예: 6개월 / 12개월 / 18개월]`
- 주요 집행 항목: `[개발 인건비 / 클라우드 / 실증 운영 / 사용자 모집 / 홍보 / 장비 연동 검증]`

투자 요청의 핵심 논리는 다음과 같다.

1. 초기 개발비보다 실증과 안정성 확보 비용이 크다.
2. 해양 안전, 지역관광, 디지털 전환이라는 공공 목적을 동시에 달성한다.
3. 실증 후 민간 구독, 제휴, 지역 연계로 자립 가능한 수익 구조를 만든다.

### 15.1 외부 API 의존성

위험: 제조사 API 정책 변경, 인증 만료, 응답 불안정  
대응: provider별 어댑터 구조, 캐시, 실패 fallback, 재시도 로직

위험: 위험도 결과를 절대값으로 오해할 수 있음

위험: 로그, 위치, 장비 데이터가 민감할 수 있음

위험: 기능이 많아 개발이 분산될 수 있음  
대응: 기록, 날씨, 연동, 공유의 4대 핵심 축 중심으로 단계화

## 17. 결론

Divergram은 다이버의 기록, 안전, 연동, 공유를 한 번에 해결하는 전문 플랫폼이다. 특히 해양 레저 안전과 지역 관광 활성화라는 공공적 가치가 분명하며, 단계적 실증과 민간 확장이 모두 가능한 구조를 갖춘다.

공공기관 투자 관점에서 Divergram의 강점은 다음과 같다.

1. 공공성: 안전 정보와 지역 해양관광 정보를 함께 다룬다.
2. 실증성: 앱과 API 구조가 명확해 파일럿 검증이 쉽다.
3. 확장성: 민간 구독과 제휴로 자립형 사업화가 가능하다.
4. 데이터성: 다이빙 로그, 위치, 날씨, 장비 데이터를 구조화할 수 있다.

따라서 Divergram은 단순 앱 개발이 아니라, 해양 레저 디지털 인프라 구축 사업으로서 공공기관 투자 유치에 적합하다.
