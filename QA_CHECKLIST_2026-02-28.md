# Divergram QA Checklist (2026-02-28)

## 1) 탐색(Explore)
- [x] 게시물/릴스/사람/리조트 탭 검색 동작 확인
- [x] 80% 스크롤 시 자동 추가 로딩 동작 확인(게시물/릴스 탭)
- [x] 썸네일 우상단 미디어 아이콘(사진/영상) 표시 확인
- [x] 영상 hover 자동재생/마우스아웃 정지 확인

## 2) 피드(Home)
- [x] 릴스(영상) 게시물 피드 제외 확인
- [x] 작성자 + 버디 `외 N명` 표기 확인
- [x] 시간 표기 규칙 확인(방금작성/시간전/일전)

## 3) 등록/수정 폼(Create/Edit)
- [x] UI 밀도/크기 간소화 확인
- [x] 리조트 `@` 입력 시 추천창 표시 확인
- [x] 스쿠버/테크니컬 기체 선택(공기/나이트록스/헬리옥스) 확인
- [x] 나이트록스/헬리옥스 선택 시 농도(%) 입력 노출 확인
- [x] 기체 농도 필수값 검증 추가

## 4) 지도(Map)
- [x] `/location` 마커 클러스터링(줌아웃) 동작
- [x] 다크모드 Google Map 스타일 적용

## 5) 테마(Dark/Light)
- [x] 핵심 페이지 대비색 1차 점검(Feed/Explore/Profile/Messages/Notifications/Search)
- [ ] 세부 컴포넌트 2차 픽셀 단위 점검(잔여)

## 메모
- 이번 턴에서는 기능 동작 안정화/필수 검증 로직까지 반영.
- 2차 테마 픽셀 QA는 실제 디바이스별 스크린샷 기준으로 후속 진행 권장.
