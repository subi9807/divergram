오픈소스 라이선스

사이트 구현에 사용된 라이브러리와 폰트는 각 프로젝트의 라이선스와 고지 조건을 따릅니다.

LibrariesJS / CSS
FontsWebfont
NoticeAttribution

사용된 라이브러리

Pretendard webfont, 브라우저 기본 API, SVG 아이콘을 사용합니다. 외부 리소스가 추가될 경우 해당 라이선스를 따릅니다.

라이선스 고지

각 라이브러리의 저작권 고지, 제한 사항, 재배포 조건은 원 저작권자의 안내를 우선합니다.

배포

오픈소스 구성요소는 프로젝트의 목적에 맞게 사용되며, 필요 시 별도 NOTICE 파일을 추가할 수 있습니다.
